Analisis Kelemahan & Rekomendasi
1. Masalah Overwrite (Penimpaan)
Kelemahan: Setiap kali script dijalankan, file syslog.txt di dalam folder backup-log akan tertimpa. Anda akan kehilangan histori backup sebelumnya.
Solusi: Tambahkan timestamp (penanda waktu) pada nama file backup agar setiap hasil backup bersifat unik.

2. Lokasi File yang Statis (Hardcoded)
Kelemahan: Script mengasumsikan syslog.txt berada di folder yang sama dengan script. Jika script dijalankan dari direktori lain (misal via cron job), script akan gagal menemukan file tersebut.
Solusi: Gunakan path absolut atau variabel untuk menentukan lokasi folder log.

3. Tidak Ada Kompresi
Kelemahan: File log biasanya berukuran besar. Menyalin file mentah tanpa dikompresi akan memboroskan ruang penyimpanan (disk space) dengan cepat.
Solusi: Gunakan tar atau gzip.

4. Izin Akses (Permissions)
Kelemahan: File sistem seperti syslog seringkali hanya bisa dibaca oleh user root atau adm. Jika script dijalankan oleh user biasa tanpa sudo, proses cp akan gagal tanpa penanganan error yang spesifik.\
