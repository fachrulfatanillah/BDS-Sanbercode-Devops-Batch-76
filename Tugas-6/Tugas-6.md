1.	Lakukan Update & Upgrade di vps ubuntu
Melakukan pembaruan dan upgrade pada server, ini adalah langkah pertama yang wajib dilakukan setiap kali masuk ke server baru agar sistemnya aman dan paket aplikasinya terbaru. Untuk melakukan pembaharuan menggunakan perintah:

```bash
sudo apt update && sudo apt upgrade -y
```
